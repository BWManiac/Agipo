Design Notes

Tab Bar

  • • 5 tabs: Tools, Inputs, Config, Connect, Test
  • • Tools tab active by default
  • • Active tab: white text + primary border-bottom
  • • Inactive tabs: slate-400 text, hover → white

Search

  • • Search icon left, clear button right (when has text)
  • • Filters both toolkit names and tool names
  • • Highlights matching text in yellow
  • • Shows "(X matches)" count in toolkit header

Toolkit Groups

  • • Collapsible with chevron indicator
  • • Shows toolkit name + tool count
  • • First toolkit expanded by default
  • • Border + subtle background

Tool Items

  • • Tool name (font-medium) + schema counts badge
  • • Description (2-line max, truncated)
  • • Hover: bg-primary/10, left border, "Click to add" hint
  • • Click → adds step to workflow
