Props

Prop Type Description
tool ComposioTool Tool data (id, name, description, schemas)
onAdd () => void Called when user clicks the item

Computed Values

Value Derived From
inputCount Object.keys(tool.inputSchema.properties).length
outputCount Object.keys(tool.outputSchema.properties).length

Behavior

  • • Click → calls onAdd() → creates WorkflowStep
  • • Hover → shows "Click to add step" hint
  • • Hover → schema counts turn primary color
  • • Description truncated at 2 lines

IPO Model: Why "X in, Y out" Matters

The input/output counts preview the IPO model (Input-Process-Output) that's central to workflow building. Users need to know:

  • → Inputs (X in): How many fields need to be mapped from previous steps or runtime inputs
  • → Outputs (Y out): How many fields are available to map to subsequent steps
