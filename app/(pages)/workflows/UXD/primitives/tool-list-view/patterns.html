List View - Control Flow Patterns

How control flow primitives (branch, parallel, loop, foreach) appear in list view. Uses indentation, grouping, and badges.

Simple Sequential

.then(A).then(B).then(C)

1
Step A
2
Step B
3
Step C

Simple vertical stack with arrows between cards.

With Data Mapping

.then(A).map(...).then(B)

1
Scrape Job
Out: title, company
2
Generate Email
In: jobTitle, companyName

Clickable badge between cards shows mapping. Click to edit.

List View Limitations

When List View Struggles

  • • Deeply nested control flow (branch inside loop inside parallel)
  • • Many parallel branches (>3-4 gets crowded)
  • • Complex data flow with multiple sources
  • • Non-linear execution paths

When to Suggest Canvas View

  • • User adds nested control flow
  • • Workflow exceeds 10 steps
  • • Multiple parallel branches
  • • User explicitly wants spatial layout
