How Profile Update Works

  1. 1. User clicks "Update profile" (pencil icon or context menu)
  2. 2. Confirmation dialog explains what will happen
  3. 3. Session starts with existing profile (already logged in)
  4. 4. Toolbar shows "Updating" badge so user knows changes will save
  5. 5. User makes changes (log into new sites, update settings)
  6. 6. User clicks "End & Save Changes"
  7. 7. Profile updated with new browser state
