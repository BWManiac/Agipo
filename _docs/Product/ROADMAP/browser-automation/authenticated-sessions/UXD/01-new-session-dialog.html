UXD: New Session Dialog

Updated dialog with 3 profile options: No profile, Use existing, Create new.

When "Create new" is selected, user enters a profile name. Session will save as that profile when ended.

New Browser Session

Start a new cloud browser session

Leave empty to start with a blank page

How it works: Log into sites during your session. When you end the session, all cookies and login state will be saved to this profile. Next time, select this profile to start already logged in.

States to implement:

  • 1. "No profile" selected - just URL input
  • 2. "Use existing" selected - show profile dropdown
  • 3. "Create new" selected - show name input
  • 4. Loading state while creating
  • 5. Error state if name taken
