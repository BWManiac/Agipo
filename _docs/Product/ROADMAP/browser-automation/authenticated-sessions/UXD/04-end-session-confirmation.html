UXD: End Session Confirmation

Shown when user clicks "End Session" on a session with a profile. Confirms that the profile will be saved.

End Session & Save Profile?

Your browser state will be saved to the profile "linkedin-work"

What will be saved:

  • Cookies & login sessions
  • localStorage data
  • Authentication tokens

Next time you use this profile, you'll start already logged in.
