Profile Types Explained:

Saved Session

Browser state saved on Anchor. Includes cookies, localStorage, logged-in sessions. Start already authenticated.

Credentials

Username/password stored locally. Browser can auto-fill login forms, but you still need to complete login flow.

May Expire

Profile hasn't been used in 30+ days. Session cookies may have expired. You might need to re-login.
