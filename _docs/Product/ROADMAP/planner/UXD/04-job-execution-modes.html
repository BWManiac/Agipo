How It Works

2

2

Not Sure Which to Choose?

Here's a simple rule of thumb:

  • • Choose Chat Template if you want to be notified and have a conversation about the task
  • • Choose Execute Workflow if you want the task done automatically without your involvement
  • • You can always change the execution mode later if your needs change
