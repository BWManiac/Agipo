## Design System

All mockups use the **v2-minimal** design language:

- **Framework:** Tailwind CSS
- **Colors:** Monochrome with amber/blue accents
- **Icons:** Lucide icons
- **Component Base:** shadcn/ui patterns

### Color Coding

| Element | Color | Meaning |
|---------|-------|---------|
| Tables | Blue (`text-blue-500`) | Structured data |
| Documents | Amber (`text-amber-500`) | Unstructured content |
| RAG Active | Green (`bg-green-100`) | RAG enabled and indexed |
| RAG Indexing | Amber (`bg-amber-100`) | Currently indexing |
| RAG Error | Red (`bg-red-100`) | Indexing failed |
