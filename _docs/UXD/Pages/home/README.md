# Home Page UXD

This folder contains UXD mockups and planning documents for the Home page.

---

## Files

| File | Description |
|------|-------------|
| `home-page-plan.md` | Product planning document with goals, layout overview, and user flows |
| `home.html` | HTML mockup of the home page |

---

## Purpose

The Home page serves as the main dashboard where users can:
- See active agents and tasks
- Monitor live activity
- Access quick actions (create agent, browse marketplace)
- View agent metrics and performance
