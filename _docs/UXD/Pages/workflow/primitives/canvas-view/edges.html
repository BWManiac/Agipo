Canvas View - Edge Types

Edges connect nodes and represent data flow. Different edge types for different purposes.

Edge Types

Sequential Edge (.then)

Step A
Step B

Simple connection, animated to show data flow direction

.then(stepA).then(stepB)

Data Mapping Edge (.map)

Scrape URL
out: title, company
title → jobTitle
Generate Email
in: jobTitle

Shows field mapping, gradient indicates transformation

.map({ jobTitle: { step: scrape, path: 'title' } })

Branch Edges (.branch)

Branch
IF
ELSE
Path A
Path B

Dashed for conditional, labels show which path

.branch([[cond, pathA], [() => true, pathB]])

Parallel Edges (.parallel)

FORK
Previous Step
A
B
C
JOIN
Next Step

All paths run simultaneously, join waits for all

Edge States (Runtime)

Edges change appearance during workflow execution:

Pending

Gray, no animation

Running

Blue, animated pulse

Completed

Green, solid

Error

Red, striped pattern

Edge Interactions

Click Edge → Open Mapping Modal

Clicking an edge between nodes opens the data mapping UI where users configure field mappings.

Drag Handle → Create Edge

Dragging from output handle to input handle creates a new edge (and prompts for mapping configuration).
